## ‚öôÔ∏è Image Processing & Performance Standards
To ensure optimal website performance and user experience, all assets in this repository undergo a strict processing workflow before being uploaded:

* **Next-Gen Formats:** Images are converted to `AVIF` or `WebP` formats to provide superior compression and quality compared to traditional JPEGs or PNGs.
* **Manual Optimization:** Assets are compressed using tools like [Squoosh](https://squoosh.app/) to find the perfect balance between crisp visual quality and minimal file size.
* **Metadata Scrubbing:** Unnecessary EXIF data is stripped from the files to further reduce payload sizes, while preserving relevant copyright/creator tags where legally required.
